About the Project

What is Promptle?

Promptle is a web-based guessing game inspired by Wordle, but designed to remove one of the biggest limits of traditional puzzle games: a fixed set of topics and answers. Many Wordle-style games rely on a hard-coded database, which means the experience eventually becomes repetitive once players have seen the available content. Promptle addresses that problem by combining structured category-based guessing with AI-generated puzzle creation, allowing the game to stay replayable and flexible over time.

Players can choose a topic, start a round, and try to identify the hidden answer by making guesses and reading the feedback returned from each guess. The game preserves the familiar satisfaction of narrowing down the correct answer through hints, but extends that model by allowing existing topics, user-driven topic input, and dynamically generated puzzles. OpenAI's API is used to generate new game content, which makes it possible to support a much wider range of subjects than a static topic list alone could provide.

Beyond the core gameplay loop, Promptle includes user accounts and profiles, support for multiplayer play, and a design intended to grow into additional AI-assisted game modes in the future. The overall goal of the project is to create a browser-based game that is replayable, customizable, and social, while still keeping each round clear, structured, and accessible to players who enjoy category-based puzzle games.

Design Commitment

Accessibility

Promptle is being developed with accessibility in mind. The site aims to support keyboard navigation, readable color contrast, clear heading structure, descriptive labels for interactive elements, and responsive layouts across devices. Our goal is to keep the experience usable and understandable for as many players as possible while continuing to improve accessibility throughout development.
